Model Settings

Configure Random Forest model parameters for optimal performance

Model Parameters

Number of decision trees in the forest (10-500). More trees = better accuracy but slower training.
Maximum depth of each tree (3-50) or unlimited. Check "Unlimited" for no depth limit.
Minimum samples required to split a node (2-20). Higher values prevent overfitting.
Minimum samples required at each leaf node (1-20). Higher values create simpler models.
Note: Changing these parameters will require retraining the model with the new configuration. This may take a few moments depending on the amount of data.

Parameter Guidelines

Recommended for Small Datasets (<100 samples)
  • n_estimators: 50-100
  • max_depth: 5-10
  • min_samples_split: 5-10
  • min_samples_leaf: 2-5
Recommended for Large Datasets (>100 samples)
  • n_estimators: 100-300
  • max_depth: 10-30
  • min_samples_split: 2-5
  • min_samples_leaf: 1-2
